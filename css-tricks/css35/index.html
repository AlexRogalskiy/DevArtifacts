Understanding CSS clip

Imagine, you need to clip some image, yeap?

But what actually does clip coordiantes mean?

Let's imagine these coordinates like this

To start with, set the start point

A and D mean top and left coordinates of start point

Then take width and height of the future clipped area

And do some math for B and C

B = D + Clip Width C = A + Clip Height

Looks like more clearly, hah? That's all!
