Numscrubber.js

Let's change values of input numbers by dragging the mouse left & right

Lightweight and easy to use. No dependencies.

Only 1.1 kb minified

Download Tweet

Examples

Default input number


<input type="number" value="2" min="0" max="40" />

Default input number with Numscrubber.js


<input data-numscrubber type="number" value="2" min="0" max="40" />

Custom input number


<input class="custom_input" type="number" value="2" min="0" max="120" />

Custom input number with Numscrubber.js


<input data-numscrubber class="custom_input" type="number" value="2" min="0" max="120" />

Custom input number with step


<input class="custom_input" type="number" value="-20" min="-50" max="120" step="10"/>

Custom input number with step and Numscrubber.js


<input data-numscrubber class="custom_input" type="number" value="-20" min="-50" max="120" step="10"/>
