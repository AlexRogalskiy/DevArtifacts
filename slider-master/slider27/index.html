  • A simple responsive static slider when no need for interaction or accessibility.
  • A slider You can customize as you like
    (e.g. same, various or full heights).
  • You only have to set a few Sass variables:
  • Your prefered class name
    $pcs-class-name,
  • Your prefered effect between
    translate or fade
    $pcs-effect,
  • the number of slides
    $pcs-slides-count,
  • the pause duration
    $pcs-timing-pause,
  • the transition duration and effect
    $pcs-timing-transition
    and $pcs-timing-function,
  • if You want to display the slides indexes
    $pcs-index-display,
  • and if You want a pause on hover
    $pcs-hover-pause.
