@include centerEl;

With this SCSS mixin you'll be able to horizontally AND vertically center blocks of text dynamically, with ease

To use it, just @include centerEl

You can set some properties through parameters too, like the left position, width and text-align.

Defaults: $left: 50%, $width: 50%, $align: center

But you don't have to horizontally align if you don't want to. Just include the mixin like this...

@include centerEl($left: 95%, $align: left);

Or you can include the mixin like this...

@include centerEl($left: 5%, $width: auto, $align: left);
