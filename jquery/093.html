Instagram-like filters with JavaScript

There is a powerful image manipulation library called Caman.js that can create all kinds of image effects with the canvas element. Note that this example will only work when run through localhost and not opened directly.

Code

// Initialize some variables var dropdown = $('#filters'), canvas = $(); dropdown.change(function(){ // Create a canvas element and insert it into the page var tmp = $('<canvas>').insertAfter(dropdown).hide(); // Get the current value of the dropdown var filter = this.value; Caman(tmp[0], 'assets/img/Katy_Perry.jpg', function(){ // Remove the canvas element from the previous operation canvas.remove(); canvas = tmp.show(); // If such a filter exists, use it if(filter in this){ this[filter]().render(); } }); }); // Show the regular image on load dropdown.trigger('change');
