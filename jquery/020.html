Define Repeated Functions Once

If you are after better performance from your code, you have to be careful when setting up event listeners. It is better to define a function once and pass it by name as an event handler:

Code

$(document).ready(function(){ function showMenu(){ alert('Showing menu!'); // Doing something complex here } $('#menuButton').click(showMenu); $('#menuLink').click(showMenu); });

HTML

<button id="menuButton">Show Menu!</button> <br /> <br /> <a href="#" id="menuLink">Show Menu!</a>
