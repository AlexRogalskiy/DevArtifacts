More about jQuery deferreds

Knowing what deferreds are and how they are best used is a great tool to your web development toolbox. Here is a small example:

Code

// Define a new class - JSON_Reader (note that if it weren't for the example // we could replace this entire class with a call to $.getJSON) function JSON_Reader(name){ var d = new $.Deferred(); $.ajax({ url: 'assets/misc/' + name + '.json', dataType: 'json', success: function(data){ d.resolve(data); }, error: function(){ d.reject(); } }); // You should return a promise object, so that // third party code can attach event handlers: return d.promise(); } // Let's use it var one = new JSON_Reader('1'); one.done(function(d){ log('Data received:', d); }); one.fail(function(){ log('The file does not exist!'); });
