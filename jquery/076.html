Upgrade jQuery with CSS3 animations

CSS3 transitions and animations are smoother and faster than the ones possible with JavaScript alone, but are not available in every browser. Use the jQuery animate enhance plugin to upgrade the library to use CSS3 animations when possible.

Code

// We have included the plugin in the page, so now the animate // method will automatically make use of the CSS3 capabilities of this browser $('#animate').click(function(){ $('#square').animate({ width:200, height:200, left:40, top:20 }); });

HTML

<button id="animate">Animate</button> <div id="square" style="width:80px;height:80px; position:absolute;background-color:white;"></div>
