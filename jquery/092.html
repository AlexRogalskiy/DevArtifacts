Work with retina graphics

To make your websites presentable on devices with high-dpi screens, you need to use higher resolution imagery. One technique is to swap the regular image with a higher resolution one with a jQuery plugin like jQuery-Retina:

Code

// Call the plugin on images you want to replace. The only // requirement is that you set a data attribute on the image. If this // is a retina device, the data-retina image will be loaded instead. $('#nature').retina();

HTML

<!-- The data-retina attribute points to an image that is twice the width and height of the src --> <img id="nature" src="http://farm5.staticflickr.com/4032/5077495230_7d9f6b1121_n.jpg" data-retina="http://farm5.staticflickr.com/4032/5077495230_7d9f6b1121_z.jpg" />
