Use classical inheritance in JavaScript

Do you know that there are JavaScript libraries that give you classes, constructors and inheritance much like with any regular object oriented language? Here is an example with the Class library, from the creators of jQuery:

Code

// Using the JavaScript inheritance class: var Animal = Class.extend({ init: function(name){ this.name = name; }, sound: function(snd){ log(this.name + ' said ' + snd + '!'); } }); var Cat = Animal.extend({ sound: function(){ this._super('meow'); } }); var Dog = Animal.extend({ sound: function(){ this._super('bark'); } }); var kitty = new Cat('Snowball'); var doggy = new Dog('Frankenweenie'); kitty.sound(); doggy.sound();
