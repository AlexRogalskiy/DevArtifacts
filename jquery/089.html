jQuery and Web Workers

Web worker threads are separate processes from the page which you can do computationally intensive operations in. There is a small wrapper library that integrates them with jQuery as a plugin.

Code

// Initialize the worker var worker = $.worker({ file:'assets/misc/worker.js' }); worker.progress(function(){ // Will output the same message that we sent it log('Received message:', this.data); }); worker.postMessage('Hello there!'); /* The contents of worker.js: self.onmessage = function(e) { self.postMessage(e.data); self.close(); }; */
