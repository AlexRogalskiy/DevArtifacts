jQuery and cookies

Reading and writing cookies is cumbersome when done with JavaScript. However, there are jQuery plugins that make it easier. One of them is jQuery.cookie:

Code

// Using the jQuery.Cookie Plugin $.cookie('password', '12345', { expires: 7 }); log('Saved! Refresh this page to test it'); // Try reopening this tab, deleting the line above // and uncommenting the line below to test it // log('My password is:', $.cookie('password'));
