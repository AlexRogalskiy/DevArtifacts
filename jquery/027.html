Single use event listener

Sometimes, you need to bind an event listener that should be executed only once. jQuery's got you covered:

Code

var press = $('#press'); // There is a method that does exactly that, the one(): press.one('click',function(){ alert('This alert will pop up only once'); }); // What this method does, is call on() behind the scenes, // with a 1 as the last argument: // press.on('click',null,null,function(){alert('I am the one and only!');}, 1);

HTML

<button id="press">Press me!</ul>
