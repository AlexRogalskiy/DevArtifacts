Find the geographic location of an IP address

There are services online that can tell you the city and country of an IP address. Here is how to use one of them:

Code

// Define variables var ip = '', // you can optionally put an ip address here api = 'http://freegeoip.net/json/' + ip + '?callback=?'; $.getJSON(api, function(r){ log('How is the weather in ' + r.city + ', ' + r.country_name + '?'); });
