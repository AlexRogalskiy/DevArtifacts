Event simulation

That plugin you like doesn't have a proper api? No problem, you can always simulate a click on the buttons it listens to:

Code

var press = $('#press'); // Just a regular event listener: press.on('click',function(e, how){ how = how || ''; alert('The buton was clicked ' + how + '!'); }); // Trigger the click event press.trigger('click'); // Trigger it with an argument press.trigger('click',['fast']);

HTML

<button id="press">Press me!</ul>
