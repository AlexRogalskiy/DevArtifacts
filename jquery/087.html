Work with money and currencies

Many web apps nead to work with and convert between multiple currencies with up-to-date exchange rates. There is an easy micro-library that makes this much easier.

Code

// First, fetch the exchange rates $.getJSON('assets/misc/rates.json',function(data){ fx.rates = data.rates; fx.base = data.base; // Then, start converting log("100 USD in EUR:", fx(100).from("USD").to("EUR").toFixed(2) ); log("2 GBP to USD:", fx(2).from("GBP").to("USD").toFixed(2) ); log("123 EUR to RUB:", fx(123).from("EUR").to("RUB").toFixed(2) ); });
