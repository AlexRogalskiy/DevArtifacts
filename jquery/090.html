Mask input fields

You may wish to allow only specific groups of characters to be entered in a text field. Here is how to do it.

Code

// Set up some input masks $("#date").mask("99/99/9999"); $("#phone").mask("(999) 999-9999"); $("#tin").mask("99-9999999"); $("#ssn").mask("999-99-9999"); // Custom definition $.mask.definitions['h'] = "[A-Fa-f0-9]"; // allow only hash values to be entered $("#color").mask("#hhhhhh");
