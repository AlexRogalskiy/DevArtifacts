Avantgrid

While we wait for Flexbox to be widely supported. Avantgrid is a mobile first responsive grid sytem utilising the property display:inline-block. This is by no means finished and is still a little rough round the edges, if you have any suggestions please feel free to get involved on Github.

Nested

Nesting is pretty simple, just add another .grid container inside a .block-X-X element. Take note all examples on this page have a padding of 1em, which isn't included in the _avantgrid.scss or grid.css so nested elemnts will look different.

H-Align

One of the nicest features of inline-block is being able to center and element. To do this change the class of the container from .grid to .grid-center or .grid-right to align to the right. Note: you should only use center or right for non-complete rows.

V-Align

The deafult vertical alignment is top, to mimic that of float. @extend .block-X-Xand then add the property vertical-align:middle or vertical-align:bottom depending on what you need.

Justify

All the cool kids seem to be doing nice things with text-align:justify and display: inline-block at the moment, so thought it best to add to the grid system. For this you must change the container class to .grid-justify and use or @extend the class .justify-X-X on the inner element instead of .block-X-X
