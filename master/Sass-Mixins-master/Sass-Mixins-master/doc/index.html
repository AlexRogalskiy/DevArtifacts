    x-at-least

    @mixin x-at-least($device-width) { ... }
    View source

    Description

    Add Breakpoint greater than the specified device-width.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $device-width—noneNumber—none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Output

      @media screen and (min-width: <device-width>) {
     <content>
  }

    Example

    Note: Please use the CSS unit `em` for device-width in order to stay
responsive.
```css
  .selector {
    @include x-at-least(40em) { width: 60%; }
  }
```

    x-linear-gradient

    @mixin x-linear-gradient($direction: to bottom, $fallback: #ccc, $from: #ccc, $to: #aaa) { ... }
    View source

    Description

    Generates a linear gradient for a given element with a fallback color.

    Note: By default this linear-gradient-mixin encourages people to use the latest CSS-syntax for gradients.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $direction

    Either an angle value or any of to bottom, to right, to top or to left

    		String or Angleto bottom
    $fallback—noneColor#ccc
    $from—noneColor#ccc
    $to—noneColor#aaa

    Output

      background-color: <fallback>;
  background-image: -webkit-gradient(linear, <direction - old converted>, from(<from>), to(<to>));
  background-image: -webkit-linear-gradient(<direction - converted>, <from>, <to>);
  background-image:         linear-gradient(<direction>, <from>, <to>);

    Example

    .selector {
  @include x-linear-gradient('to bottom', #ccc, #ddd, #bbb);
}

    x-multiple-colored-gradient

    @mixin x-multiple-colored-gradient($args...) { ... }
    View source

    Description

    This mixin creates (endless) multiple color stops in gradients just with one call for the mixin.

    Note: This mixin does not define a fallback-color for your background as it is likely you want to add an image or something. Please specify one by yourself.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $args—noneArglist—none

    Output

      background-image: -webkit-linear-gradient(top, <stops[1]>, <stops[2]>, ..., <stops[n]>);
  background-image:         linear-gradient(to bottom, <stops[1]>, <stops[2]>, ..., <stops[n]>);

    Example

    .selector {
  @include x-multiple-colored-gradient(
      'top',
      #f22 0%,
      #f2f 15%,
      #22f 30%,
      #2ff 45%,
      #2f2 60%,
      #2f2 75%,
      #ff2 90%,
      #f22 100%
  );
}

    x-rem

    @mixin x-rem($property, $values, $default-font-size: 16px) { ... }
    View source

    Description

    This mixin enables you to use the CSS3 value rem, which lets you define property sizes based on the root element's font-size.

    Outputs properties that use rem with a px fallback. It also takes px values and converts them to rem.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $property—noneString—none
    $values—noneList—none
    $default-font-size

    optional

    		Number16px

    Output

      <property>: <parsed value>px;
  <property>: <parsed value>rem;

    Example

    .selector {
  @include x-rem(font-size, 1.3);
  @include x-rem(padding, 20px);
