jQuery keyboard shortcuts

Every modern web app worth its salt has to have keyboard shortcuts. Here is a neat plugin that will help you with that:

Code

// Set up your key bindings key('ctrl+s', function(){ log('Saving!'); // Prevent the default browser event return false; }); key('ctrl+r', function(){ log('No reloads for you!'); return false; }); key('ctrl+o', function(){ log('Showing an open dialog'); }); key('d', function(){ log('Deleting messages!'); }); log('Try presing some keys!');
