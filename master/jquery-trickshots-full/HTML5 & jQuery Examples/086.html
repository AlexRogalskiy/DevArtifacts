Turn textboxes into rotating knobs

With the jQuery Knob plugin, you can emulate a real rotating control. It even supports touch which makes it a good fit for mobile web apps.

Code

// Trigger the plugin with an input text field $('#dial').knob();

HTML

<input type="text" id="dial" value="23" data-min="0" data-max="100" />
