Minimum Dom Touches

Despite all the advances in JavaScript performance, manipulating the dom is still expensive and should be kept to a minimum. This is especially important when inserting lots of elements on the page at once:

Code

// Bad //var elem = $('#elem'); //for(var i = 0; i < 100; i++){ // elem.append('<li>element '+i+'</li>'); //} // Good var elem = $('#elem'), arr = []; for(var i = 0; i < 100; i++){ arr.push('<li>element '+i+'</li>'); } elem.append(arr.join(''));
