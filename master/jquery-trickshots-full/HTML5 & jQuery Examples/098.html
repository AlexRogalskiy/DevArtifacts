Handling file drag and drop

Another cool feature that modern browsers provide is the ability to drag and drop files from your desktop into the browser. Here is how to do it with the Filereader.js library:

Code

// Call and configure the plugin var pics = $('#pics'); $('body').fileReaderJS({ on:{ load: function(e, file){ var img = $('<img>'); img.load(function() { img.css({ maxWidth:150, maxHeight:150 }); pics.append(img); }); img.attr('src', e.target.result); }, beforestart: function(file){ // Allow only images return /^image/.test(file.type); } } });

HTML

Drop an image on the page and it will appear here. <div id="pics"></div>
