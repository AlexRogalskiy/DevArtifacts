jQuery callback lists

jQuery's $.Callback object gives you a way to manage a large number of callback functions easily. Here is how it is used:

Code

// First example - Using the callbacks object. // Create a new callbacks object var cb = $.Callbacks(); // Add a callback cb.add(function(message){ log(message); }); // Add one more cb.add(function(){ log('---'); }); // Trigger the callbacks $('#btn1').click(function(){ clear(); // Output three lines cb.fire("Line 1"); cb.fire("Line 2"); cb.fire("Line 3"); }); // Second example - Using flags // Will stop executing callbacks on false var cflags = $.Callbacks('stopOnFalse') // Again add the clear function cflags.add(clear); cflags.add(function(){ // Is the checkbox checked? return $('#checkBox').is(':checked'); }); cflags.add(function(){ log('Checkbox is checked!'); }); // Trigger the callbacks $('#btn2').click(function(){ cflags.fire(); }); // Handle the clearing of the screen function clear(){ log.clear(); } $('#clear').click(clear);

HTML

<button id="btn1">1</button> <button id="btn2">2</button> <input type="checkbox" id="checkBox" checked /> <button id="clear" style="float:right">Clear</button>
