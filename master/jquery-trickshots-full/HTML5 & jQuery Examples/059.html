Create a custom pseudo selector

You can extend jQuery with your own selectors. Here is how it's done:

Code

// To create a custom selector, add it to the $.expr.pseudos property $.expr.pseudos.icontain = $.expr.createPseudo(function(arg) { return function(elem) { return (elem.textContent || elem.innerText || jQuery(elem).text() || '') .toLowerCase() .indexOf(arg.toLowerCase()) >= 0; }; }); // Here is how it is used: log( 'Paragraphs that have the string "lorem" in them:', $('.lipsum:icontain(lorem)').length );
