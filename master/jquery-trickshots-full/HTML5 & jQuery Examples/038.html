The simplest AJAX request

jQuery offers a shorthand method for quickly loading content into an element with AJAX:

Code

var contentDivs = $('.content'); // Fetch the contents of a text file: contentDivs.eq(0).load('assets/misc/1.txt'); // Fetch the contents of a HTML file, and display a specific element: contentDivs.eq(1).load('assets/misc/1.html #header');

HTML

<p class="content"></p> <p class="content"></p>
