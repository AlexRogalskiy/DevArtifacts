Get your IP with jQuery

Did you know you can get your public IP address with one line of JS? There is a free service that offers this for you:

Code

$.get('http://jsonip.com/', function(r){ log(r.ip); }); // For older browsers, which don't support CORS // $.getJSON('http://jsonip.com/?callback=?', function(r){ log(r.ip); });
