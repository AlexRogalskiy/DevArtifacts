Generate QR Codes with jQuery

QR codes encode textual information in an image which you can scan with your smartphone. Do you know you can generate them with jQuery?

Code

// Simply call the plugin and pass some options: $('#qr').qrcode({ // Width an height of the code width: 200, height: 200, // QR code color color: '#000', // Background color, null for transparent background bgColor: null, // The encoded text text: 'http://www.google.com/' });

HTML

<div id="qr" style="height:220px;padding:30px;background-color:#fff; text-align:center"></div>
