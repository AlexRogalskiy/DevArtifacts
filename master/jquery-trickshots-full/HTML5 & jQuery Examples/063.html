The jQuery $.proxy() method

jQuery sets the context of every callback function to the originating element. With $.proxy, you can work around this:

Code

// Listen for button clicks $('#toggle').click( $.proxy(function(){ // "this" would normally point to the button. // With $.proxy(), we are overriding it to point // to the football this.toggle(); }, $('#football') ));

HTML

<button id="toggle"> Show/Hide Ball</button><br /> <img id="football" src="assets/img/ball.png" style="display:none;" />
