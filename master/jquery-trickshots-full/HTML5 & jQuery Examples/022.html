Detach elements when doing complex modifications

Modifying a DOM element requires the page to be repainted which can be expensive. If you want to squeeze every bit of performance, you can try detaching the element from the page when doing intense modifications on it.

Code

var i = 0, iterations = 1000; // Modifying in place var elem = $('#elem'); timer('In place'); for(i=0; i < iterations; i++){ elem.width(Math.round(100*Math.random())); elem.height(Math.round(100*Math.random())); } timer_result('In place'); var parent = elem.parent(); // Detaching first timer('Detached'); elem.detach(); for(i=0; i < iterations; i++){ elem.width(Math.round(100*Math.random())); elem.height(Math.round(100*Math.random())); } elem.appendTo(parent); timer_result('Detached');

HTML

<div id="elem" style="background:blue"></div>
