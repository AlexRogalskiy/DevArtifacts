Speed up your site with a script loader

Browsers block the rendering of the page when they encounter a script include. This can be remedied with a script loader like head.js:

Code

// Lets load a few libraries and a js file // located in the assets folder of this demo head.js( 'http://cdnjs.cloudflare.com/ajax/libs/jqueryui/1.9.2/jquery-ui.min.js', 'http://cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/2.2.2/bootstrap.min.js', 'assets/misc/include.js', function(){ // Everything is loaded! Let's print the message defined in include.js: log(message); } );
