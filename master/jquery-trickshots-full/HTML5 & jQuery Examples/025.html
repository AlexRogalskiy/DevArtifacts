Assign a "JS" class to the html element

Modern web apps are increasingly reliant on JavaScript. Here is one technique that makes it easy to show certain elements on your page only when JavaScript is enabled. Hit Run below, and see the source of this page for an example.

Code

// That is all there is to it! $(document).ready(function(){ $('html').addClass('JS'); });

This is a message that is only shown when JavaScript is available. This way you would hide elements or areas of the site that don't make sense without JS.
