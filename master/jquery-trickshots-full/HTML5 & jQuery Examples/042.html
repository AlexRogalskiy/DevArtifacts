Access weather information

Open Weather Map provides free weather information, that you can use for free through their JSON API. Here is an example:

Code

// Request weather data: var api = 'http://openweathermap.org/data/2.1/find/name?q=paris,france&callback=?'; $.getJSON(api, function(r){ // This will always give the current picture log(r.list[0].name + ', ' + r.list[0].sys.country); log(r.list[0].main); // Temperatures are in kelvin, subtract 273.15 to convert to Celsius, // or use the formula (kelvin*9‚ÅÑ5 - 459.67) for Fahrenheit });
