# Base Easing Functions

The base easing functions could be expressed with strings that contain `easing name` and `direction` delimited by `.` or could be accessed directly on `mojs.easing` object:

```javascript
  // ...
  easing: 'cubic.in',
  // or
  easing: mojs.easing.cubic.in
  // ...
```

The full list of base functions:

```javascript

  'linear.none'

  'ease.in'
  'ease.out'
  'ease.inout'

  'sin.in'
  'sin.out'
  'sin.inout'

  'quad.in'
  'quad.out'
  'quad.inout'

  'cubic.in'
  'cubic.out'
  'cubic.inout'

  'quart.in'
  'quart.out'
  'quart.inout'

  'quint.in'
  'quint.out'
  'quint.inout'

  'expo.in'
  'expo.out'
  'expo.inout'

  'circ.in'
  'circ.out'
  'circ.inout'

  'back.in'
  'back.out'
  'back.inout'

  'elastic.in'
  'elastic.out'
  'elastic.inout'

  'circ.in'
  'circ.out'
  'circ.inout'

```
