BBMEnterprise.Media.DataConnection.DataReceiver

This class manages the data receiver of a data connection.

A data receiver is created by the data connection when a new incoming data channel is detected. All incoming data associated with the data channel is handled by the receiver.

Based on the data type of the receiver, the client could configure the data receiver to appropriate receiving modes with BBMEnterprise.Media.DataConnection.DataReceiver#configure.

configure(cfg, chunkSize)

Configures the receiving mode of the data receiver.

NOTE:

  • Can only be called in the handler of BBMEnterprise.Media.DataConnection#event:dataReceived.

  • STREAM data type can only be configured as STREAM mode.
