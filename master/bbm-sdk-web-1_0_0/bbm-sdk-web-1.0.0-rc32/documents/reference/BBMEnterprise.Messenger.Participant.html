BBMEnterprise.Messenger.Participant

Provides a structure to encapsulate the details related to a chat participant. Each instance of a Participant is immutable.

Properties

isAdmin  boolean

This will be set to true when the participant is an admin in the chat. This only applies when the 'isOneToOne' field is false for the chat. This value will never change.

regId  string

The BBM Registration ID of the participant. It is the responsibility of the application to relate the regId of a participant to a user within the ecosystem.
