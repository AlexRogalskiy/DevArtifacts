BBMEnterprise.Media.DataConnection.DataSender

This class manages a data sender of a data connection

A data sender is used to send arbitrary data of any size over a data channel.

With the data sender, user can stream data of any (known or unknown) size with maximum flexibility. With no buffering on the receiver side, data is also received with low latency.

The receiver of this data channel will be typed as BBMEnterprise.Media.DataConnection.DataReceiver.DataType.STREAM.

send(data) → Promise

Sends a given string or Uint8Array over the data channel.

Parameter

Name Type Optional Description

data

(String or Uint8Array)

 

data in the form of a string or Uint8Array, to be sent.
Returns

Promise 

A Promise that resolves when the data is completed sending and could reject with the following errors:

  1. BBMEnterprise.Media.Exceptions.InvalidArgumentError if called with unknown data type.

  2. BBMEnterprise.Media.Exceptions.InvalidStateError if send failed due to connection errors or aborted due to termination of the data connection.
