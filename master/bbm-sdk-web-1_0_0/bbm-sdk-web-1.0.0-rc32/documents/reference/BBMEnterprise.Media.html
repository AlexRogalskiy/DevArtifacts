BBMEnterprise.Media

This class manages the media calls and data only connections.

This class depends on BBMEnterprise and an instance is created by the BBMEnterprise after it is successfully setup (see BBMEnterprise#setup).

NOTE, Media calls are only supported on browsers with WebRTC implementation. Certain browsers may require running with the adapter.js See

createDataConnection(callee, metaData) → Promise

Makes an outgoing request to establish a data connection.

Parameters

Name Type Optional Description

callee

Object

 

the callee BBMEnterprise.Media.Callee of the outgoing request.

metaData

Object

 

an object sent to the callee as meta data of the request.

It is typically used to describe the purpose of the data connection.
Throws

BBMEnterprise.Media.Exceptions.NotSupportedError if the media features are not supported on the platform.

BBMEnterprise.Media.Exceptions.InvalidArgumentError if the arguments are invalid.

BBMEnterprise.Media.Exceptions.InvalidStateError if the call cannot be made due to transport state (e.g. no connection).

Returns

Promise 

a Promise, which is resolved to a DataConnection object BBMEnterprise.Media.DataConnection and may reject to the following errors:

  1. BBMEnterprise.Media.Exceptions.TimeoutError if the data connection fails to be requested due to timeout. e.g. if the server failed to resolve the callee's identity.

  2. BBMEnterprise.Media.Exceptions.InvalidStateError if the data connection fails to be requested due to connection state. e.g. if the connection to server has dropped during the request.

makeCall(callee, callOPtions) → Promise

Makes an outgoing voice/video call.

Parameters

Name Type Optional Description

callee

Object

 

the callee BBMEnterprise.Media.Callee of the outgoing call.

callOPtions

Object

 

the call options BBMEnterprise.Media.CallOptions of the outgoing call.
Throws

BBMEnterprise.Media.Exceptions.NotSupportedError if the media calls are not supported on the platform.

BBMEnterprise.Media.Exceptions.InvalidArgumentError if the arguments are invalid.

BBMEnterprise.Media.Exceptions.InvalidStateError if the call cannot be made due to transport state (e.g. no connection).

Returns

Promise 

a Promise, which is resolved to a call object BBMEnterprise.Media.Call and may reject to the following errors:

  1. BBMEnterprise.Media.Exceptions.NotSupportedError if the requested call options cannot be satisfied, e.g. video device is unavailable on the platform when the video option is enabled.

  2. BBMEnterprise.Media.Exceptions.PermissionDeniedError if the media fails to be requested due to permission error. e.g. if the user denied the request to allow camera or mic to be accessed.

  3. BBMEnterprise.Media.Exceptions.TimeoutError if the media fails to be requested due to timeout. e.g. if the server failed to resolve the callee's identity.

  4. BBMEnterprise.Media.Exceptions.InvalidStateError if the media fails to be requested due to connection state. e.g. if the connection to server has dropped during the request.
