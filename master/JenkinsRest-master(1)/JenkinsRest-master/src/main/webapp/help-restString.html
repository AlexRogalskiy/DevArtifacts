A string that is the template for the data being sent. The values that should be replaced are all caps, underscores for spaces, and prefixed and suffixed with a %. The following values are replaced:

  • %BUILD_ID%: the build id/number.
  • %JOB_NAME%: the name of the job.
  • %BUILD_MESSAGE%: the message at the end of the build.
  • %DATETIME%: A timestamp of the scheduled build.
