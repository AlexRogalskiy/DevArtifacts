CSSVR

CSSVR is an experiment that simulates browser support for creating VR experiences using media queries and CSS transforms. You can read about the project on my website. The source is available on GitHub.

Demos

Please orientate your viewport to landscape and, if your browser supports it, enable fullscreen. Then insert your device into a VR headset.

For the best experience, add these demos to your homescreen.

  • Hello world — A bare-bones demo.
  • Skybox — CSS 3D animation with CSS :hover states and a skybox.
  • Scene interaction — Interactive demo with click events and :hover states.
