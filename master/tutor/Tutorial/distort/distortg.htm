PhotoLine Tip: Eine verzerrte Schrift
1. Beginnen Sie mit einem Bild Ihrer Wahl, z. B. dieser hübschen Blume.

2. Schreiben Sie den gewünschten Text darauf.

3. Wählen Sie nun das Ebenenwerkzeug aus der Werkzeugleiste und setzen Sie den Bearbeitungsmodus auf Verzerren. Dies können Sie entweder dauerhaft über die Werkzeugeinstellungen zum Ebenenwerkzeug (auch im Kontextmenü möglich) oder kurzzeitig durch Drücken der Tasten Strg und Alt erreichen.

4. Verschieben Sie nun die oberen Ecken der Textebene mit der Maus.

5. Das Ergebnis sollte etwa so aussehen.
