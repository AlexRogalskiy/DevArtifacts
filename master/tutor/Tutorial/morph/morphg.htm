PhotoLine Tip: Ein Bild in ein anderes Morphen
1. Beginnen Sie mit zwei Bildern, die als zwei ebenen eines Bildes vorliegen müssen, z. B.
         
Falls Sie zwei getrennte Bilder haben, ziehen Sie eines der Bilder mit dem Ebenenwerkzeug (eventuell bei gedrückter Alt-Taste) auf das andere Bild.

2. Wählen Sie dann Filter/Verformen/Morphen.

Sie sehen den Morphdialog. Auf der linken Seite wir das Quellbild angezeigt, auf der rechten Seite das Zielbild. Die Vorschau in der Mitte des Dialoges zeigt die Morphschritte. Um diese zu sehen, müssen Sie den Schieberegler unter der Vorschau bewegen.
Zuerst müssen Sie einige Pfeile im Quellbild plazieren. Um einen Pfeil zu setzen klicken Sie in das Quellbild und ziehen Sie die Maus bei gedrückter Maustaste auf den gewünschten Endpunkt. Immer wenn Sie im Quellbild einen Pfeil setzen, erscheint im Zielbild eine Kopie dieses Pfeiles. Aktivieren Sie dann Edit Modus und schieben Sie dann den Pfeil im Zielbild auf den Bildinhalt auf den der Quellpfeil abgebildet werden soll.
Wenn es einfacher ist, können Sie auch zuerst den Pfeil in das Zielbild setzen und dann im Quellbild verschieben.

3. Setzen Sie nun die gewünschte Anzahl von Zwischenbildern im Feld Schritte.

4. Drücken Sie OK und PhotoLine 32 wird eine Animation erzeugen.

5. Wählen Sie Web/Ebenen animieren um die Animation anzusehen. Sie können die Animation im PLD- oder GIF-Format speichern.
