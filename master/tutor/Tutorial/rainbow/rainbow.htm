PhotoLine Tip: A rainbow
1. You have an image of a landscape (or any other).

2. Create a rainbow blend. It should look like this:

Circle-Blend, HIS-Mode, start transparent, then the rainbow color and the end transparent.

3. Now create the blend from the middle bottom point in the image to the top. You will get a rainbow like this:
