PhotoLine Tip: Eine Visitenkarte anlegen
1. Sie möchten eine Visitenkarte anlegen. Dazu legen Sie zuerest en neues Dokument in der Größe der gewünschten Visitenkarte an. Wählen Sie dazu Datei/Neu/Neues Dokument....

Hier sollte die Größe der Visitenkarte in mm eingegeben werden. Sollte in den Feldern eine andere Einheit stehen, so kann durch einen Klick mit der rechten Maustaste aus einem Popup die gewünschte Einheit ausgewählt werden.

Sie liegen mit einer Größe von 85 mm auf 54 mm nicht schlecht. Sollten Sie jedoch spezielles Visitenkartenpapier verwenden, so stellen Sie bitte die darauf angegebenen Werte ein.

2. Sie erhalten nun ein leeres Dokument. Hier sollten nun zuerst die gewünschten Textinformationen eingegeben werden. Wählen Sie hierzu das Textwerkzeug aus der Werkzeugleiste.

Stellen Sie in den Ebeneneinstellungen die gewünschte Schriftart ein.

3. Klicken Sie nun in das leere Dokument und geben Sie den gewünschten Text ein. Hierbei sollten Sie sich die spätere Gestaltung überlegen.

4. Sie können natürlich auch, wie in unserem Beispiel einen Rahmen um die Visitenkarte zeichnen. Benutzen Sie hierzu die Grafikprimitive. Wir haben das einfache Rechteck gewählt.

Stellen Sie hier die Füllfarbe auf transparent (Deckung 0) und die Rahmenfarbe auf schwarz. Die Liniendicke stellen wir auf 3 Pixel.

5. Nun können wir den Namen noch etwas verschönern indem wir einen 3D-Schatten dazu anlegen. Wir wählen dazu die Funktion Effekte/Effekte/3D-Schatten.... Durch die Aktivierung der Vorschau sehen wir sofort den eingestellten Schatten im Bild.

6. Nun wollen wir die Visitenkarte noch mit einem kleinen Bild verzieren. Dazu öffnen wir das gewünschte Bild. Dann wählen wir das Ebenenwerkzeug aus der Werkzeugleiste.

Damit kann das Bild auf die Visitenkarte geschoben werden. Um ein unabsichtliches Verschieben zu verhindern, sind Einzelbilder allerdings durch ein Schloß im Ebenendialog vor dem Verschieben geschützt.

Sie können diese "Verriegelung" durch einen Linksklick darauf ausschalten. Alternativ können Sie vor dem Verschieben die Alt-Taste drücken (dadurch wird die Schloßfunktion kurzzeitig ausgeschaltet).

7. Mit der Maus kann das Bild plaziert werden. Durch Ziehen an den Ecken kann die Größe des Bildes geändert werden. Um das Bild proportional zu skalieren muß dabei die Alt-Taste gehalten werden.

8. Jetzt ist die Visitenkarte fertig und sollte im PLD-Format gespeichert werden. Es ist wichtig das PLD-Format zu wählen, weil nur so die Texte und auch der 3D-Schatten später wieder geändert werden können.

9. Nun muß natürlich die Visitenkarte noch ausgedruckt werden. Wählen Sie zur richtigen Einstellung des Ausdrucks Datei/Druckeinstellungen.... Hier stellen Sie den Druckmodus auf Etikett und die Breite und Höhe auf das gewünschte Format (85x54 mm im Beispiel).

Die Ränder stellen Sie gemäß Ihrem Visitenkartenpapier ein oder falls Sie auf Normalpapier drucken auf die Druckränder des Druckers. Wenn Sie das Dokument jetzt nochmal im PLD-Format sichern, merkt sich PhotoLine 32 die Einstellungen.
