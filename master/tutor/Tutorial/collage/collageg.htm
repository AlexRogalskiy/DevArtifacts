PhotoLine Tip: Collagen - Bildteile zusammenfügen
1. Beginnen Sie mit einem beliebigen Bild.

2. Um die Arbeit zu erleichtern sollten Sie beim Photografieren darauf achten einen möglichst einfarbigen Hintergrund mit möglichst hohem Kontrast zum Bild zu wählen.
Wählen Sie aus der Werkzeugleiste das Automask Werkzeug.
Stellen Sie die Werkzeugeinstellungen etwa so ein:

Ca. 30% Toleranz, Löcher füllen angewählt, Weicher Rand 50.

3. Klicken Sie jetzt in die obere rechte Ecke des Bildes. Falls zu wenig selektiert wurde, können Sie mit gedrückter Umschalt Taste die Automaske vergrößern. Falls zu viel selektiert wurde läßt sich die Automaske bei gedrückter Strg Taste verkleinern.

4. Danach muß die Makse noch mit Werkzeuge/Maske/Invertieren invertiert werden (nur, wenn Sie mit der Automaske den Hintergrund angewählt haben).

5. Öffnen Sie nun ein Bild, daß der Hintergrund werden soll.

6. Aktivieren Sie das Lassowerkzeug und ziehen Sie das maskierte Bild auf das
Hintergrundbild. Abhängig von Ihren Einstellungen erhalten Sie nun eine Selektion oder eine neue Ebene im Hintergrundbild Fenster. Wenn Sie eine Selektion haben, ziehen Sie das Bild auf den gewünschten Platz und setzen Sie die Selektion mit einem Doppelklick ab. Dann drücken Sie Strg+D um die Selektion zu verwerfen. Wenn Sie eine neue Ebene erhalten haben, aktivieren Sie das Ebenen Werkzeug und schieben Sie die neue Ebene an die gewünschte Stelle. Dann wählen Sie Ebenen/Auf Hintergrund reduzieren.
