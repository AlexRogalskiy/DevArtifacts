PhotoLine Tip 6: Lassen Sie Ihren Text brennen!
 
1. Beginnen Sie mit einem neuen Bild, z. B. 200 x 100 pixel groß, RGB.  
 
2. Öffnen Sie, falls nötig, den Ebenendialog. Sie können den Dialog öffnen, indem Sie 
Anzeige/Ebenen anwählen. Dann erzeugen Sie eine neue Ebene, mit der gleichen Größe und dem gleichen Bildtyp wie die Hintergrundebene. Wählen Sie hierzu Ebene/Neue Ebene und drücken Sie OK.  
  
3. Wählen Sie nun das Vektortext Werkzeug aus der Werkzeugleiste (klicken Sie auf das Text Icon und halten Sie die Maustaste gedrückt, bis eine Auswahl erscheint. Wählen Sie daraus das Vektortext Werkzeug) und tippen Sie Ihren Text in den Werkzeugoptionen Dialog. Klicken Sie auf Zeichensatz um die Größe und die Schriftart einzustellen, z. B. Arial 26 Punkte. Wählen Sie Setzen um den Text zu plazieren. 

 

4. Sie können nun den Text beliebig plazieren, beachten Sie aber beim Verschieben Strg/Alt zu drücken, da sonst die Maske verloren geht. 
5. Wählen Sie nun das Verlaufswerkzeug aus der Werkzeugleiste.  

6. Erzeugen Sie hier einen linearen Verlauf von schwarz nach gelb.  

 

7. Setzen Sie den Verlauf in das Bild. 
 
8. Wechseln Sie nun wieder auf die Hintergrundebene (klicken Sie auf die obere Ebene im Ebenendialog) 

9. Erzeugen Sie eine dunkelrote Farbe (R180, G0, B0) für die Vordergrundfarbe und wählen Sie eine unscharfe Pinselspitze (etwa 7 Pixels). Damit malen Sie etwas, wie das untenstehende Bild: 
 
Um das zu erreichen, wählen Sie eine zweite Farbe hellrot (R255, G0, B0) und etwas gelb. 

10. Wählen Sie nun das Fingerwerkzeug  aus der Werkzeugleiste. Nehmen Sie eine 
Pinselspitze von etwa 5 Pixels und verwischen sie das Feuer:
