PhotoLine Tip: Zwei Bilder auf ein Blatt
1. Sie haben einen Text gescant und von einer anderen Vorlage einen Fußballspieler.

2. Nun möchten Sie den Text auf das Bild mit dem Fußballspieler setzen. Damit der Text Platz hat, muß zuerst das Bild des Fußballspielers vergrößert werden (nicht skaliert). Dazu aktivieren Sie das Textbild und wählen Sie Ebene/Ebenengröße verändern....

Hier können Sie die Größe des Bildes ändern. Sie können wählen, ob Sie die neue Größe eingeben wollen oder die Größenänderung. Mit den Pfeilen unter dem Bild können Sie festlegen, an welcher Position des neuen Bildes das alte Bild liegen soll.

3. Nun gibt es mehrere Möglichkeiten um den Fußballer auf das Textbild zu bekommen
- Sie können über dem Fußballer ein Lasso anlegen und dieses einfach auf das Textbild ziehen. Wenn Sie die Maustaste über dem Text loslassen erhalten Sie eine Selektion mit dem Fußballer. Nun können Sie den Fußballer an die gewünschte Stelle schieben. Dann können Sie mit einem Doppelklick den Fußballer in das Textbild setzen oder Ebene/Konvertiere Selektion in Ebene anwählen um eine neue Ebene mit dem Fußballspieler zu erhalten.
- Sie können aber auch das Ebenenverschiebewerkzeug anwählen und den Fußballer auf den Text schieben. Sie erhalten beim Textbild eine Selektion. Danach müssen Sie zum Lassowerkzeug wechseln und können dann wie oben beschrieben verfahren.
- Sie können das Fußballerbild aktivieren und durch Strg+C kopieren. Dann das Textbild aktivieren und mit Strg+E oder Bearbeiten/Als Selektion einfügen den Fußballer als Selektion einfügen.
