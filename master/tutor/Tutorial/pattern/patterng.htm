PhotoLine Tip: Ein Muster erzeugen
1. Sie möchten ein Muster für den Einsatz mit PhotoLine 32 erzeugen. Beginnen Sie mit einem Bild Ihrer Wahl. Es kann gescannt oder digital aufgenommen sein.

2. Suchen Sie sich einen Ausschnitt, der wiederholbar gemacht werden soll und ziehen Sie ein Rechtecklasso um diesen Ausschnitt. Danach öffnen Sie das Kontextmenu (rechte Maustaste drücken) und wählen Beschneiden.

3. Wählen Sie nun das Werkzeug Filter/Sonstige/Ebeneninhalt verschieben. Setzen Sie hier die horizontale und die vertikale Verschiebung jeweils etwa auf die halbe Bildgröße und wählen Sie Verschobene Bereiche kopieren.

4. Sie erhalten ein Bild, beim dem jetzt der Rand passt. An den mit Pfeilen gekennzeichneten Linien aber gibt es eine Kante.

5. Diese Kante muss jetzt entfernt werden. Dazu benutzt man am Besten den Kopierpinsel und den Finger

Das Ergebnis sollte etwa so aussehen:

6. Nachdem das Muster fertig ist, muss es noch in die Musterliste eingefügt werden. Öffnen Sie hierzu die Farbliste (Ansicht/Listen/Farbliste).

Hier können Sie das Muster mit dem Ebenenwerkzeug

auf einen freien Platz in der Liste, über ein vorhandenes Muster oder auf dem Schiebebalken neben den Mustern ziehen.
Normalerweise ist die Hintergrundebene im Ebenendialog vor dem Verschieben geschützt.

Dies kann auf einen Klick auf die entsprechenden Symbole beseitigt werden. Alternativ kann auch vor dem Verschieben die Alt-Taste gedrückt und gehalten werden. Dies löscht die Sperre kurzzeitig.
