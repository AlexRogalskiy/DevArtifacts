PhotoLine Tip: Pillars in PhotoLine 32
1. We want to create some pillars. Therefore we create RGB-image with 150x300 pixel.

2. Now take the blend tool from the toolbar and setup a linear blend from black to middle grey. It should look like this:

3. Now activate the mode "Edit Mask".

4. Create a blend in the mask like this.

5. Now leave the "Edit Mask" mode.

6. Setup a nice foreground color and select the fill tool from the tool bar. Fill the pillar and you will get pillars like this:
