PhotoLine Tip: Ein Ball mit Schatten für Knöpfe
1. Beginnen Sie mit einem neuen Bild, z. B. 80x80 Pixel groß und RGB.

2. Erzeugen Sie ein Kreislasso (Klicken Sie auf das freie Lasso Icon und halten Sie die linke Maustaste gedrückt. Es erscheint ein Popup, aus dem Sie das Kreislasso auswählen können. Drücken Sie die Alt-Taste auf der Tastatur und ziehen Sie im Bild einen Kreis auf)

3. Wählen Sie eine beliebige Farbe aus der Farbliste aus und setzten Sie damit die
Vordergrundfarbe, z. B. rot.
4. Wählen Sie nun Werkzeugool/Lasso/Füllen Vordergrund.

5. Wählen Sie nun Verlauf aus der Werkzeugleiste.

6. Erzeugen Sie hier einen Kreisverlauf von schwarz, voll transparent, nach schwarz voll deckend.

7. Setzen Sie den Verlauf jetzt auf den Ball.

8. Wählen Sie nun Effkete/Spezialeffekte/Schatten. Setzen Sie den Winkel 45, Intensität 50, Breite 8, Unscharf 10 und drücken Sie OK.

9. Sie können jetzt noch einen kleinen unscharfen Pinsel auswählen und einen weißen Punkt auf den Ball setzen.

10. Drücken Sie zum Schluß Ctrl + D um das Lasso zu löschen und speichern Sie das Bild.
