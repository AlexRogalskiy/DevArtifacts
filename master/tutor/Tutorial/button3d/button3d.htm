PhotoLine Tip: Create a 3D-button in PhotoLine32
1. Start with a image of your choice, best is a texture, for example 100 x 50 pixel in size, RGB color.


2. Create a text layer, for example with the Text Tool from the tool bar.

3. Type your text. Be awear that you have to select the text first if you want to edit size or type of the font.

4. Select now Effects/Effects/Inner Emboss... Select at Special the switch Hide Original and Hard Light.

5. To save the image for a web side, you have to select the function Layer/Reduce To Background Layer first.
