PhotoLine Tip: Erzeugen Sie einen 3d-Knopf mit PL32
1. Beginnen Sie mit einem Bild Ihrer Wahl, am Besten mit einer Textur, z. B. 100 x 50 Pixel, RGB.

2. Erzeugen Sie eine Textebene, z.B. über die Funktion Ebene/Neue Ebene....

3. Schreiben Sie Ihren Text. Beachten Sie, dass es nötig ist den Text zu selektieren, wenn Sie die Größe oder die Schriftart ändern wollen.

4. Wählen Sie jetzt Effekte/Effekte/Hervorhebung innen.... Wählen Sie hier unter Spezial den Schalter Original ausblenden und Hartes Licht.

5. Um den Knopf für eine Webseite speichern zu können müssen Sie jetzt noch die Funktion Ebene/Auf Hintergrundebene reduzieren anwählen.
