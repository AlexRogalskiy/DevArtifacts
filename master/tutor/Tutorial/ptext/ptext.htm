PhotoLine Tip: Text along curves - path text
1. You want to create path text for your image

2. Create a vector layer to setup the path for the text. You can use the vector create tool or one of the vector primitives to do this.

3. Select the text tool now. Move the mouse cursor to the path. When you reached the path (you see a path text cursor) click to start text entering.

4. Now you should setup the vector layer invisible in the layer dialog.
