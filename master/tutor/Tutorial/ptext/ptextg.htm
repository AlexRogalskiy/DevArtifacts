PhotoLine Tip: Text entlang von Kurven - Pfadtext
1. Sie haben ein Bild, das Sie originell beschriften möchten.

2. Erzeugen Sie eine Vektorebene, die die Lage des Textes vorgeben soll. Sie können dazu das Vektoreingabewerkzeug oder eines der Vektorelemente aus der Werkzeugleiste benutzen.

3. Wählen Sie nun das Textwerkzeug aus der Werkzeugleiste. Plazieren Sie den Mauszeiger über den vorgegebenen Pfad. Wenn Sie den Pfad erreicht haben (zu erkennen an der geänderten Form des Mauszeigers) klicken Sie in das Bild um den Text einzugeben.

4. Wenn Sie den Text eingegen haben, schalten Sie im Ebenendialog die Vektorebene auf unsichtbar.
