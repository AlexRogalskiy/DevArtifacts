PhotoLine Tip: Dynamische Animation: Wie man ein rotierendes Bild erzeugt
1. Beginnen Sie mit einem Bild Ihrer Wahl, z. B.

2. Wählen Sie Web/Dynamische Animation.

3. Klicken Sie hier auf das i-Symbol um die Animationsinformationen aufzurufen

Stellen Sie hier die Länge auf die gewünschte Rotationsdauer, z.B. 5 Sekunden.

4. Ziehen Sie jetzt das Bild aus dem rechten Bereich des Dialogs in die Zeitleiste. Hier "ziehen" Sie am rechten Rand des Bildes, so dass die ganze Zeitleiste bedeckt ist.

5. Ziehen Sie jetzt den kleinen Schieberegler unter der Sequenz in die Mitte und klicken Sie mit der rechten Maustaste um im Kontextmenü Position übernehmen anzuwählen.

6. Im Dialog Sequenzeigenschaften, den Sie öffnen indem Sie auf das i mit dem kleinen Filmstreifen klicken, stellen Sie die Breite des mittleren Zeitpunkts auf -100% (war 100%).

7. Schon ist die Animation fertig. Sie können sie testen, indem Sie im Dialog Dynamische Animation links unten auf den kleinen Pfeil klicken. Jetzt können Sie die Animation in einem geeigneten Format speichern, z.B. GIF oder SWF (Flash). Beachten Sie bitte, dass die Webexport-Funktion keine Animationen speichern kann.
