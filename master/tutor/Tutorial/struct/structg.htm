PhotoLine Tip: Ein Bild mit Struktur
1. Sie können mit einem neuen Bild (z. B. 80 x 80 Pixel groß, RGB) oder mit einem Bild Ihrer Wahl beginnen.
2. Wählen Sie Maske bearbeiten (klicken Sie auf das rechte Icon unter der Vordergrund/Hintergrundfarbe)
3. Wählen Sie Filter/Qualität/Rauher. Setzen Sie die Größe auf 3x3, 80% und
drücken Sie OK. Verlassen Sie den Maske bearbeiten Modus (Klicken Sie nochmal auf das Icon)
4. Wählen Sie Effekte/Effektfilter/Hervorheben. Setzen Sie den Winkel auf 45, die Höhe auf 45 Grad und die Breite auf 3. Drücken Sie OK.
Wenn Sie mit einem leeren Bild gearbeitet haben, so haben Sie nun ein graues Papier.
5. Wenn Sie die Farbe ändern möchten, wählen Sie
Werkzeug/Farben/Farbart/Sättigung. Wählen Sie hier den Einfärben Knopf. Setzen Sie die Intensität auf 0, und die Sättigung auf 100. Zum Ändern der Farbe verschieben Sie den Farbart Regler.
