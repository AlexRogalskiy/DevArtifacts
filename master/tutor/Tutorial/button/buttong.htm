PhotoLine Tip: Ein Knopf in klassischer Technik
1. Beginnen Sie mit einem neuen Bild, z. B. 100 x 100 Pixel groß, RGB.

2. Erzeugen Sie ein Kreislasso (klicken Sie auf das freie Lasso Icon und halten Sie die
Maustaste gedrückt bis eine Auswahl erscheint. Wählen Sie hier das Kreislasso. Drücken Sie die Alt-Taste während Sie den Kreis aufziehen)

3. Wählen Sie nun das Verlaufswerkzeug aus der Werkzeugleiste.

4. Erzeugen Sie einen linearen Verlauf von einer dunklen Farbe zu einer hellen.

5. Setzen Sie den Verlauf in das Bild.

6. Drücken Sie Strg+D um das alte Lasso zu löschen und erzeugen Sie ein zweites Kreislasso etwas kleiner als das erste. Zeichnen Sie nun den Verlauf in entgegengesetzter Richtung.

7. Jetzt noch die Beschriftung und fertig ist der Knopf
