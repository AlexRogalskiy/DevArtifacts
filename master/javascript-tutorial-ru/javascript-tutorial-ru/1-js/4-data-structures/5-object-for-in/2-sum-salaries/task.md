# Сумма свойств

Есть объект `salaries` с зарплатами. Напишите код, который выведет сумму всех зарплат.

Если объект пустой, то результат должен быть `0`.

Например:

```js
"use strict";

var salaries = {
  "Вася": 100,
  "Петя": 300,
  "Даша": 250
};

//... ваш код выведет 650
```

P.S. Сверху стоит `use strict`, чтобы не забыть объявить переменные.
