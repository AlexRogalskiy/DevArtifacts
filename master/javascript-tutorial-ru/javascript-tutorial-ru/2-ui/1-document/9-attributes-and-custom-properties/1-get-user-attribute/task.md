# Получите пользовательский атрибут

1. Получите `div` в переменную.
2. Получите значение атрибута `"data-widget-name"` в переменную.
3. Выведите его.

Документ:

```html
<body>

  <div id="widget" data-widget-name="menu">Выберите жанр</div>

  <script>
    /* ... */
  </script>
</body>
```
