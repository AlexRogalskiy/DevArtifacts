# Часики

Создайте компонент "Часы" (Clock).

Интерфейс:

```js
var clock = new Clock({
  elem: элемент
});

clock.start(); // старт
clock.stop(); // стоп
```

Остальные методы, если нужны, должны быть приватными.

При нажатии на `alert` часы должны приостанавливаться, а затем продолжать идти с правильным временем.
