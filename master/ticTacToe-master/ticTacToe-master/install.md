# Instructions

* untar the tarball
* cd <untarred-directory>
* install deps
```
npm install
```

* configure the mongodb url in ./server/config/config.js
```
#start server
node app
```

+ load homepage
  curl -i "http://localhost:8080"
+ test server
  node test/test.js
